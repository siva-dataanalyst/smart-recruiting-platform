# 🎯 Smart Recruiting Platform

An intelligent resume screening system that automatically ranks candidates using Machine Learning.

## 📌 Features
- 📄 Resume Upload and Text Extraction
- 🔍 Job Description and Skill Matching
- 🤖 ML Scoring using TF-IDF and Cosine Similarity
- 📊 Candidate Ranking Dashboard
- 📈 Data Visualizations

## 🛠 Tech Stack
- Python
- Streamlit
- PyPDF2
- Pandas
- Scikit-learn
- Matplotlib

## 🚀 How to Run
pip install -r requirements.txt
streamlit run app.py
